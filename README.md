# ebook-capture

macOS 전자책 뷰어에서 원하는 화면 영역을 지정한 뒤, 페이지를 자동으로 넘기며 PNG 파일로 순차 저장하는 데스크톱 유틸리티입니다.

개인 학습 및 개인 보관 목적의 빠른 화면 캡처를 돕기 위해 제작되었습니다.

## Features

- 드래그 방식의 캡처 영역 선택
- 오른쪽 방향키 입력을 통한 자동 페이지 넘김
- 중복 화면 감지 기반 자동 종료
- 첫 페이지와 끝 페이지를 지정하는 범위 캡처
- 듀얼 모니터 환경 지원
- `page_0001.png`, `page_0002.png` 형식의 순차 파일 저장

## Quick Start

```bash
./mac/EbookCapture
```

1. 전자책 뷰어에서 캡처를 시작할 페이지로 이동합니다.
2. `mac/EbookCapture`를 실행합니다.
3. `저장 경로`를 지정합니다.
4. `영역 드래그 선택`으로 캡처할 페이지 영역을 지정합니다.
5. 캡처 모드를 선택합니다.
6. `캡처 시작`을 누릅니다.

캡처가 시작되면 페이지 넘김 대상 앱은 선택 영역 아래의 창으로 자동 지정됩니다. 자동 지정이 되지 않는 경우, 시작 후 3초 안에 전자책 뷰어 창을 클릭하세요.

## Capture Modes

### 자동 종료

`최대 페이지`까지 캡처하되, 같은 화면이 반복되면 마지막 페이지로 판단해 자동 종료합니다.

### 페이지 범위

`첫 페이지`와 `끝 페이지`를 기준으로 지정한 구간만 캡처합니다. 이 모드에서는 `최대 페이지` 설정을 사용하지 않습니다.

## Permissions

앱이 정상 동작하려면 macOS에서 아래 권한을 허용해야 합니다.

- 화면 캡처: `시스템 설정 > 개인정보 보호 및 보안 > 화면 기록`
- 키보드 자동 입력: `시스템 설정 > 개인정보 보호 및 보안 > 손쉬운 사용`

권한을 허용한 뒤 앱을 다시 실행하세요. 앱 안의 권한 버튼을 통해 설정 화면을 바로 열 수 있습니다.
